# Skill: Safe Implementer

Use this skill when implementing an approved plan.

## Behavior
- Keep diffs small.
- Follow existing patterns.
- Do not refactor unrelated code.
- Ask before destructive commands.
- Update tests/docs when relevant.
- Run safe validation commands when available.

## Output
- Files changed
- What changed
- Why it changed
- Tests or checks run
- Remaining risks

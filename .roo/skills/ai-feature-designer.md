# Skill: AI Feature Designer

Use this skill for AI assistant, categorization, summaries, document parsing, anomaly detection, or reporting features.

## Behavior
- Define the user problem first.
- Separate deterministic business logic from AI output.
- Require human review for financial decisions.
- Consider prompt injection, data leakage, and hallucination.
- Prefer auditable AI outputs with stored inputs, model name, timestamp, and confidence/reasoning summary where appropriate.

## Output
- AI feature goal
- User flow
- Data needed
- Guardrails
- Failure modes
- Implementation plan

# Skill: Security Reviewer

Use this skill for security-sensitive changes, especially auth, finance data, AI features, uploads, APIs, and database access.

## Checklist
- Authentication and session handling
- Authorization and RBAC
- Input validation
- Secrets management
- SQL/ORM injection risks
- File upload risks
- Audit logging
- Rate limiting
- Prompt injection risks
- Data privacy and encryption concerns

## Output
- Findings by severity
- Concrete fixes
- Files affected
- Residual risk

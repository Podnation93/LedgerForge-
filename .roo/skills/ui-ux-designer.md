# Skill: UI/UX Designer

Use this skill for dashboard, navigation, forms, layout, and product polish.

## Behavior
- Inspect existing design system/components first.
- Prefer consistent reusable components.
- Include empty, loading, error, and success states.
- Consider mobile responsiveness and accessibility.
- Keep finance dashboards clear, calm, and information-dense without clutter.

## Output
- UX goals
- Page structure
- Component plan
- States
- Accessibility notes

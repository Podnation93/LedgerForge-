# Skill: Debugger

Use this skill when investigating bugs, failing tests, crashes, or confusing behavior.

## Behavior
- Gather symptoms first.
- Inspect relevant logs/errors/tests.
- Form hypotheses.
- Make one focused fix at a time.
- Do not refactor while debugging unless the root cause requires it.

## Output
- Symptoms
- Root cause hypothesis
- Evidence
- Fix
- Prevention
